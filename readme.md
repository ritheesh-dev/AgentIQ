# 🚀 Agentic AI System with MCP (Work in Progress)

## 📌 Overview

This project focuses on building an **Agentic AI system using MCP (Model Context Protocol)** with tool-calling capabilities and dynamic reasoning using LangGraph.

## ⚙️ Current Features

* LLM with tool binding
* Basic LangGraph workflow (LLM node)
* Message state handling
* Initial MCP integration (in progress)

## 🚧 Work in Progress

* Tool execution node
* Conditional routing (agent loop)
* RAG Implementation
* MCP server integration
* Memory handling
* Multi-step reasoning

## 🧠 Goal

To build a fully functional **autonomous AI agent** that:

* Dynamically decides when to use tools
* Maintains context across steps
* Integrates with MCP-compatible systems

## 🛠️ Tech Stack

* Python
* LangGraph
* LangChain
* Groq API
* MCP (in progress)

🟡 Currently under active development
